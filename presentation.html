What Is AngularJS, Really?

  • It is not a databinding library like Knockout, or a catchall 'glue' library like jQuery.
  • It is a full-on clientside single page application (SPA) framework. It does databinding, routing, view orchestration, depedency injection, custom HTML directives, etc.
